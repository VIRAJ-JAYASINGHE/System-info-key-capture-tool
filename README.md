# 🖥️ System Monitoring Tool (Python)

## 📖 Description

This project is a Python-based system monitoring tool that collects system performance data such as CPU, RAM, and disk usage.

The purpose of this project is to understand:

* System resource monitoring
* Automation using Python scripts
* Email-based reporting systems

> ⚠️ **Ethical Notice**
> This project is strictly for educational purposes only.
> It does NOT promote or support unauthorized data collection or user monitoring.

---

## ⚙️ Features

* 📊 CPU usage monitoring
* 🧠 RAM usage tracking
* 💾 Disk usage analysis
* 📧 Automated email reporting (using SMTP)
* ⌨️ Input event monitoring (demonstration purpose only)

---

## 🛠️ Technologies Used

* Python
* pynput
* psutil
* smtplib
* email.message

---

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/system-monitor.git
cd system-monitor
```

### 2. Install dependencies

```bash
pip install psutil pynput
```

### 3. Run the script

```bash
python main.py
```

---

## 📦 Convert to Executable (.exe)

### Step 1: Install PyInstaller

```bash
pip install pyinstaller
```

### Step 2: Build the .exe file

```bash
pyinstaller --onefile main.py
```

### Step 3: Output location

The `.exe` file will be inside the `dist/` folder.
